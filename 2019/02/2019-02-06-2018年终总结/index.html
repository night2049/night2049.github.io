2018年终总结

2019/02/05 17:31 One-minute read
Night
生活 • 年终总结

2018年，对于我个人人生中，是一个非常奇妙的节点。

这一年，我二十岁了。

二十岁代表着什么，在童年的我眼中，似乎就意味着真正的独立，可以和朋友玩个彻夜，也可以去做一些自己想做的事情。

有时候想想，人生不过短短数十载。而我已经经历过了两个十年，剩下的几个十年里，谁也不知道会发生些什么。我虽仍有期盼，但也依旧有些迷茫，自己的人生，好像有时候，并不能真正的把握在自己手中，感觉自己就像一颗石子，被时代的浪潮裹挟着前进，虽然也曾奋力逆时代而上，但却无法把控自己的走向。

时代，是一个很奇特的词汇，说起来十分轻巧，好似隔壁家今天又吃了什么一样轻松。但对于我而言，时代这个词，却如泰山般沉重。

雷军说过一句很有趣的话：站在风口上，猪也能飞起来。

最开始我觉得这句话略显好笑，有时候也对于那些看似成功的人，内心阴暗的想着，你们也只不过是站在风口上的猪罢了。

后来我却明白了，能找到并站在风口上的猪，也是必须要有些许本事的，若无压身之技，即使站在风口上，也会很快被后面的猪挤下去。

在过去的这一年里，我想了很多。

想了自己的未来该何去何从，也想了自己的生活应该怎样改进，还想了自己身边的人和事。

自己的未来，想了很久，却也未曾想出个啥，好像自己的脑袋被抽空。

生活的改进，现在唯一做的一件事情，就是换了个略大的房子。

身边的人和事，倒是思考的最多，在我而眼，身边的人更像是一面面镜子，既能照出自己丑恶的内心，也能照出人也依旧有向善的心。孔子常说三人行必有我师。我倒是觉得，人与人之间相处最大的好处，莫过于是学习对方的长处，并依照对方的短处，看看自己是不是也会犯下类似的错误，从而改进自己的言行。

当然了，对于身边的人和事，最大的感悟就是：看一个人或一件事，不要看他是怎么说，要看他怎么做。

就像那句极为经典的话一般：“论迹不论心，论心无完人”

一个人，要看他做了什么，而不是看他在想什么。

每个人的内心都会有些许阴暗，这并不妨碍他是一个好人，亦或者是一个坏人。

要看他是遵从内心的阴暗，还是内心的光明。

在这一年里，我喜欢上了一个偶像，她叫“杨超越”

这对于我来说是十分不可思议的，对于我而言，追星这种事情，是既无意义，也无好处的事情

而杨超越的出现，对于我来说，是既有意义，也有好处的偶像

我在杨超越身上学习到，一个人可以不成功，但是一定要保持一颗善良的心，即使世界被满满的恶意充斥着，也并不意味着这个世界快要完蛋了，我们也仍需要看到，还有许多人，在默默坚持着内心的美好和善良，同时也在做着能让这个世界变得更美好的事情。

同时，杨超越的乐观也影响到了我，记忆最深的一句话是在杨超越选择创作组时，黄子韬问她为什么会选择创作组，她什么也不会，不害怕老师们嫌弃她吗。而她的回答是：她是一张白纸，老师们最喜欢白纸，因为可以绘画更美好的蓝图。

曾经年少的我，和老师斗气，和家人斗气，好像需要的是证明给谁看。

老师不喜欢我，那我也就不喜欢老师，它的课我也不听，似乎就意味着我报复了老师一般。

不喜欢学习，家人非逼着学习，所以就干脆不学，就好似自己证明了自己能掌控一切一般。

而在杨超越身上，我看到了一股劲，这种劲很奇妙，是一种说不清，道不明的味道。

要是说的具体些，可能更像是戈壁滩上突然冒出来的一朵花儿，在戈壁滩的大风下摇摇欲坠，似乎随时就会被大风连根刮起，消散在天地之间一般，而你左等右等，她这朵花儿的跟，却还依旧深深的扎在那石子之下，漫天的大风虽然将她的身躯吹的摇摇欲坠，却从来没有将她的根脱离那石子一丝一毫。

而那石子，或许就是她内心的善意，有了这股善意，似乎这个世界如何残酷，却也阻挡不了那朵花带给人们的美好。

回首2018年，自己的生活虽然有磕磕绊绊，但大致而言，是不错的一年。

换了一家新公司，有了不错的同事和很nice的上司们

换了一个略大的房子，有了一个阳台，晚上无事时，看看窗外灯火璀璨的城市，脚下的车水马龙，虽然我或许不属于这座城市，但在那短短的几十秒中，好像做了一次这个城市的主人

摘掉了眼镜，可以不再透过镜片去感受这个世界的颜色

思想的境界有了些许的提升

技术上也有了些许的进步

2019年，希望自己能够保持阅读的习惯，改进自己的不足，加强自身的优势，还有就是一定要开始健身，保持一个良好的精神和身体。
