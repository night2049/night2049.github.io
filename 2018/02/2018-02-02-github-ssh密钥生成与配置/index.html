GitHub SSH密钥生成与配置

2018/02/02 02:49 One-minute read
Night
GitHub • SSH密钥

打开Git Bash

ssh-keygen -t rsa -b 4096 -C "你的邮箱"

直接回车确认即可，也可根据提示进行设置。
添加ssh key+

eval $(ssh-agent -s)

ssh-add ~/.ssh/id_rsa

然后打开：

C:\Users\Your UserName\.ssh\id_rsa.pub

复制文件所有内容然后打开GitHub，在个人设置中点击SSH and GPG keys，然后点击New SSH key。设置你的这个ssh key的标题，然后将文件内容复制到key中，点击Add SSH key即可完成操作。
