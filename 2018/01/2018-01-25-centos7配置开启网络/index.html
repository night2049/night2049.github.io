centos7配置开启网络

2018/01/24 17:31 One-minute read
Night
Linux • CentOS

cd /etc/sysconfig/network-scripts/

使用

ls

会显示该目录下的所有文件，找到以ifcfg开头的第一个文件

vi ifcfg-XXX

上方命令的xxx对应每个人的机器不一样，自己找一下对应格式的文件即可

执行上方命令后，会打开编辑器，移动光标到最后一行,按i进入编辑模式

ONBOOT=NO

修改为

ONBOOT=yes

按esc键后输入

:wq

保存该文件后执行以下命令

service network restart

即可正常使用网络
