firewalld防火墙基本操作

2018/01/24 17:33 One-minute read
Night
firewalld • Linux

firewalld是Centos7的默认防火墙，在我们使用centos中关于防火墙的操作需要经常使用，下面可以看到firewalld防火墙的基本操作。

启动： systemctl start firewalld

查看状态： systemctl status firewalld

停止： systemctl disable firewalld

禁用： systemctl stop firewalld

启动firewalld服务：systemctl start firewalld.service

关闭firewalld服务：systemctl stop firewalld.service

重启firewalld服务：systemctl restart firewalld.service

显示firewalld服务的状态：systemctl status firewalld.service

在开机时启用firewalld服务：systemctl enable firewalld.service

在开机时禁用firewalld服务：systemctl disable firewalld.service

查看firewalld是否开机启动：systemctl is-enabled firewalld.service

查看已启动的服务列表：systemctl list-unit-files|grep enabled

查看启动失败的服务列表：systemctl --failed

查看firewalld版本： firewall-cmd --version

查看firewalld帮助： firewall-cmd --help

显示firewalld状态： firewall-cmd --state

查看firewalld打开的端口： firewall-cmd --zone=public--list-ports

更新firewalld防火墙规则： firewall-cmd --reload

查看firewalld区域信息: firewall-cmd --get-active-zones

查看firewalld指定接口所属区域： firewall-cmd --get-zone-of-interface=eth0

firewalld拒绝所有包：firewall-cmd --panic-on

firewalld取消拒绝状态： firewall-cmd --panic-off

firewalld查看是否拒绝： firewall-cmd --query-panic

firewalld 开启端口操作

添加永久生效端口

firewall-cmd --zone=public --add-port=80/tcp --permanent（–permanent永久生效，没有此参数重启后失效

重新载入

firewall-cmd --reload

查看端口

firewall-cmd --zone= public --query-port=80/tcp

删除端口

firewall-cmd public --remove-port=80/tcp --permanent
